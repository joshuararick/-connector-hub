Security

Security rules for this free version.

  • Never store production API keys, tokens, passwords, refresh tokens, or private keys in this browser-only site.
  • Use placeholders until a private backend is added.
  • Host only over HTTPS.
  • Add real authentication before storing private team/customer data.
  • Use least-privilege API keys, rotate secrets, and keep backups safe.
  • Recommended backend later: server-side vault or environment variables, not localStorage.
